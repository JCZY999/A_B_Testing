# 📊 A/B Testing Analysis
Overview

This project analyzes an A/B test to evaluate the impact of two variants on user behavior and conversion performance.

## Objective
Compare Variant A (control) vs Variant B (treatment)
Measure key metrics (e.g., conversion rate, engagement)
Determine statistical significance

## Methodology
Cleaned and validated experiment data
Calculated core metrics (conversion rate, CTR, etc.)
Performed statistical testing (t-test / proportion z-test, 95% confidence level)
Results
Variant B showed a X% lift in conversion rate
Result was statistically significant / not significant (p-value: X.XX)

## Tech Stack
SQL
Python (pandas, scipy/statsmodels)
Tableau / Power BI

## Structure
data/        # raw & cleaned data  
sql/         # data extraction queries  
notebooks/   # analysis & testing  
results/     # outputs & summaries
